S1:乘客部分：
先把机场进出的人流量模拟出来吧，以首都国际机场为例：
数据引用自https://wenku.baidu.com/view/2d28cbe364ce0508763231126edb6f1aff0071f7.html?from=search
经调查，机场主流的客运机为100座级的ERJ190，客座率为83.2%（数据应用自https://www.ceicdata.com/zh-hans/china/air-passenger-load-factor），由客运量 = 航班量 航班座位数 客座率 可整理出任意小时段内进出机场的乘客数目
据资料显示，到达机场的旅客中34%乘坐出租车，离开机场中出租车的比例占13%（资料来源自https://wenku.baidu.com/view/266f4a63a22d7375a417866fb84ae45c3b35c2f5.html）（猜测可能是因为赶飞机的人比较赶时间外加出发地离地铁站较远所以出租车比例高，离开机场中赶时间的人较少且机场多有地铁及其他的公共交通，故离开机场的人中乘坐出租车离开的人比较少）
由此可计算出任意小时段内“乘坐出租车到达机场的旅客数”和“乘坐出租车离开机场的旅客数”
公式：乘坐出租车离港人数 = 离港乘客 离港做出租车率 （pas_in = round( pas_in 0.832 100 0.34)
乘坐出租车入港人数 = 入港乘客 入港做出租车率 （ pas_out = round( pas_out 100 0.832 0.13 )

S2：出租车部分

出租车送客后离开机场的情况：
该种情况下，出租车前往最近的人群集中区收益比较大，由于北京市过大，以市中心为离开机场的目的地明显不合理。经查阅，距离北京首都国际机场T2航站楼-停车场最近的人群集中区为国家会计学院，学校附近人流量大且附近有居民区，以此地为目的地可以相对较快的借到下一个乘客。
据调查，出租车每公里耗油约0.1升，现在89#汽油为6.23元，折合每公里成本0.62元。据百度地图显示，北京首都国际机场T2航站楼-停车场至国家会计学院6.6公里，驾车约13分钟
可得，空车离开机场情况下，抵达人群集中区需耗时13min，盈利-4.1元。
同时，据实地调查采访，某热心的北京出租车司机月薪7000-8000，月休八天，日均工作12h.计算后可得北京出租车司机每小时盈利约30元。

出租车送客后去机场等待接客的情况：
该种情况下，司机等待时间受“前方等待车辆”“当前到港航班数”等因素影响较大，故我们直接采用新民报提供的数据（http://mini.eastday.com/mobile/170929160648680.html），可得出租车司机在需车厂等待时间约两小时左右。
乘客目目的地以北京市中心天安门估算，据百度地图数据显示，车程为47分钟，30.2公里，打车费95元。抛去30公里的油价成本18元，司机盈利期望约78元。

总结：在理想情况下，在针对出租车司机卸下乘客—出租车司机卸下机场等待的乘客 这2h47min计算时，司机离开机场收益为市中心接客盈利30*2.5h-4.1，纯盈利70.9元。司机进入蓄车池等待时纯盈利为78元。

模型建立：
离开的情况下：统计时段离场收益 = （候场时间+候场送客时间-离场前往市区时间） 市中心每小时平均收益 - 立场油费
统计时段候场收益 = 候场送客公里数 （2.3【每公里收费】 - 0.6【每公里油费】） + 1【燃油附加费】

这样凑合整了个第一二问能交上去的模型，大体框架有了有空完善下。
